JavaScript对象按类型可以分为内置对象、浏览器对象和自定义对象
JavaScript常用的内置对象主要包括：Array对象、String对象、Math对象、Date对象
  1. Array对象：
    var 数组对象名称 = new Array(数组元素个数);或
    var 数组对象名称 = new Array()；
    使用示例：
    var fruite = new Array(2);
    fruite[0]="apple";
    fruite[1]="pear";
    定义的同时赋值：
    var 数组对象名称 = new Array(元素一,元素二,元素三);
    使用示例:
    var fruit = new Array("apple","pear","banana");

    常用方法：reverse()、sort()、join(分隔字符)、push()、splice(m,n)、concat()
  2. String对象
